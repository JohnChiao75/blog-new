本教程覆盖Minecraft Java版从环境配置、启动安装到进阶玩法的全流程，适配Windows/macOS/Linux系统，所有操作均提供官方渠道与安全指引，新手可直接按步骤操作。

## 一、前置准备：Java运行环境安装
Minecraft Java版必须依赖Java运行环境（JRE）才能启动，**游戏版本与Java版本必须严格匹配**，否则会出现启动失败、崩溃等问题。

### 1. 版本对应规则（核心）
| 游戏版本 | 推荐Java版本 | 备注 |
|----------|--------------|------|
| 1.16.5及以下 | Java 8 (64位) | 老版本模组核心适配版本 |
| 1.17-1.20.4 | Java 17 (64位) | 官方强制要求的最低版本 |
| 1.20.5及以上 | Java 21 (64位) | 新版本优化首选，向下兼容部分旧版本 |

> 提示：官方启动器会自动内置对应版本的Java，无需手动安装；第三方启动器（HMCL/PCL2）支持一键下载安装Java，新手可直接跳过手动安装步骤。

### 2. 官方下载与安装
1.  下载地址（优先选择OpenJDK，兼容性更好）：
    - Azul Zulu OpenJDK（推荐，全平台支持）：https://www.azul.com/downloads/
    - Oracle官方Java：https://www.java.com/zh-CN/download/
2.  安装注意事项：
    - 安装路径**严禁包含中文、空格、特殊符号**，建议默认路径或纯英文路径（如`D:\Java\`）
    - Windows系统务必选择64位版本，32位系统无法正常分配内存
    - 安装完成后，第三方启动器会自动检测，无需手动配置环境变量

## 二、启动器选择、下载与安装
启动器是管理游戏版本、账号、模组的核心工具，分为官方启动器与第三方启动器，以下为国内主流启动器的详细教程。

### 1. 官方启动器（Minecraft Launcher）
- 特点：官方出品，安全稳定，完美适配正版账号，支持Java版+基岩版双版本管理；缺点是国内下载速度慢，无模组管理、版本隔离等进阶功能，仅推荐纯正版用户使用。
- 官方下载地址：https://www.minecraft.net/zh-hans/download
- 安装步骤：
  1.  下载对应系统的安装包（Windows为`.exe`/`.msi`，macOS为`.dmg`，Linux为`.deb`）
  2.  双击安装包，按引导完成安装，安装路径避免中文
  3.  安装完成后打开启动器，即可进行账号登录与游戏下载

### 2. HMCL启动器（Hello Minecraft! Launcher）
- 特点：全平台支持（Windows/macOS/Linux），完全免费开源，功能全面，支持正版/离线/外置登录，一键安装游戏、加载器、Mod、整合包，自带版本隔离，新手友好，国内用户首选之一。
- 官方下载地址：https://hmcl.huangyuhui.net/download/
- 安装步骤：
  1.  下载对应系统的版本，Windows推荐下载`.exe`便携版，macOS下载`.dmg`，Linux下载`.jar`
  2.  新建一个纯英文路径的文件夹（如`D:\HMCL\`），将启动器文件放入其中（避免直接放桌面/系统盘）
  3.  双击启动器文件即可运行，首次启动会自动检测Java环境，引导完成基础设置

### 3. PCL2启动器（Plain Craft Launcher 2）
- 特点：Windows专属，界面简洁美观，操作逻辑符合国内用户习惯，下载速度快，支持一键安装、版本隔离、模组管理，免安装绿色软件，新手极度友好。
- 官方下载地址：https://afdian.net/a/LTCat（作者龙腾猫跃爱发电主页，免费发布）
- 安装步骤：
  1.  从官方发布页下载最新版压缩包`PCL2.zip`
  2.  解压到纯英文路径的文件夹中（如`D:\PCL2\`），严禁直接在压缩包内打开
  3.  双击`Plain Craft Launcher 2.exe`即可运行，首次启动会自动检测Java环境，完成初始化配置

> 安全提醒：务必从官方渠道下载启动器，切勿从第三方网站、网盘下载，避免恶意软件、盗号风险。

## 三、游戏本体与模组加载器安装
### 1. 游戏本体下载
所有启动器均支持一键下载游戏本体，操作逻辑一致，新手优先选择正式版（Release），避免快照版（Snapshot）出现兼容性问题。
- 官方启动器：左侧选择「Java版」，点击「安装」，选择需要的版本，等待自动下载完成即可。
- HMCL/PCL2：
  1.  进入启动器的「版本下载」/「下载」页面
  2.  在列表中选择想要的正式版（如1.20.1、1.21）
  3.  点击「安装」，启动器会自动完成游戏核心文件、库文件、资源文件的下载，无需手动操作

### 2. 模组加载器安装
模组（Mod）必须依赖对应的加载器才能运行，主流加载器分为4种，可根据需求选择：
| 加载器 | 特点 | 适用场景 |
|--------|------|----------|
| Forge | 模组数量最多，兼容性最强，老版本支持完善 | 玩大型科技、魔法、冒险整合包，1.12.2/1.16.5等老版本 |
| Fabric | 轻量化，启动速度快，优化更好，新版本更新快 | 玩轻量模组、优化模组，1.18+新版本首选 |
| Quilt | Fabric的分支，兼容大部分Fabric模组，修复了诸多问题，社区活跃 | 追求更好的兼容性与优化，新版本体验 |
| NeoForge | Forge的分支，专注新版本优化，兼容部分Forge模组 | 1.20.1+新版本模组体验 |

- 一键安装方法（新手首选）：
  1.  在启动器的「版本下载」页面，选择好游戏版本后，下方会出现「Mod加载器」选项
  2.  勾选你需要的加载器，推荐选择「推荐版/稳定版」，不要选择测试版
  3.  点击「安装」，启动器会自动匹配对应版本的加载器，完成一键安装，无需手动下载任何文件
  4.  安装完成后，在启动器主页选择该版本，即可启动带加载器的游戏

## 四、账号登录与正版购买
### 1. 账号登录方式
Minecraft Java版支持3种登录方式，各有不同的适用场景与限制。

#### （1）微软账号登录（正版登录）
- 前提：你的微软账号已购买Minecraft Java版授权
- 优势：可进入正版服务器、使用官方皮肤、同步云存档、参与官方活动，无功能限制
- 操作步骤：
  1.  启动器选择「微软登录」/「正版登录」
  2.  点击登录，会自动弹出浏览器授权页面，或显示授权码
  3.  登录你的微软账号，输入授权码完成授权，启动器会自动完成账号配置
  4.  登录成功后，即可启动游戏，进入正版服务器

#### （2）离线登录
- 优势：无需购买正版，无需联网，直接设置用户名即可进入游戏，适合新手试玩、单人体验
- 限制：无法进入正版验证的服务器，皮肤仅自己可见，无法使用官方云服务，联机限制较多
- 操作步骤：
  1.  启动器选择「离线登录」
  2.  设置你的游戏用户名（仅支持英文、数字、下划线，3-16个字符）
  3.  确认后即可完成配置，直接启动游戏

#### （3）外置登录
- 定义：通过第三方认证服务器完成登录，替代官方正版验证
- 优势：无需正版，可和好友联机时互相看到皮肤、进入专用服务器，支持自定义皮肤披风
- 适用场景：和好友联机、使用皮肤站服务，主流平台为LittleSkin（国内最大皮肤站）
- 操作步骤（以LittleSkin为例）：
  1.  注册LittleSkin账号：https://littleskin.cn/
  2.  进入账号设置，完成角色创建，记录你的认证服务器地址
  3.  启动器选择「外置登录」/「第三方登录」，填入认证服务器地址、你的LittleSkin账号密码
  4.  完成登录后，即可启动游戏，联机时可正常显示皮肤

### 2. Minecraft正版购买
Minecraft Java版现已与基岩版捆绑销售，一次购买可同时获得两个版本的授权，**仅官方渠道为正规安全渠道**。

#### （1）官方购买渠道
1.  Minecraft官网购买（推荐）：
    - 官网地址：https://www.minecraft.net/zh-hans/store/minecraft-java-bedrock-edition-pc
    - 步骤：登录微软账号 → 选择「Java&基岩版双版本合集」→ 完成支付（支持信用卡、支付宝等方式）→ 购买成功后，授权自动绑定到你的微软账号，无需额外激活
2.  微软商店购买（Windows用户专属）：
    - 步骤：打开微软商店 → 搜索「Minecraft Java & Bedrock Edition」→ 登录微软账号 → 完成支付 → 自动绑定账号

#### （2）购买注意事项
- 严禁购买「黑卡、共享号、代充」，此类账号均为违规获取，会被微软封禁，导致账号丢失、游戏授权收回
- 不要相信「低价正版」，官方价格固定，远低于市场价的均为违规账号
- 购买后，仅可通过微软账号登录游戏，不存在单独的游戏账号密码，切勿泄露你的微软账号信息

## 五、皮肤与披风使用
皮肤是更改玩家外观的核心方式，分为3种类型，对应不同的登录方式。

### 1. 正版皮肤（微软账号专属）
- 优势：全服务器通用，所有正版玩家均可看到，官方支持，无任何限制
- 操作步骤：
  1.  打开Minecraft官方网站，登录你的微软账号：https://www.minecraft.net/zh-hans/msaprofile/mygames/skins
  2.  选择「上传皮肤」，皮肤文件必须为png格式，尺寸为64x64（新版支持64x128），分为Steve（宽臂）和Alex（细臂）模型
  3.  上传完成后，选择对应的模型，点击保存，重启游戏即可生效
  4.  也可在官方启动器的「皮肤」页面，直接上传更换皮肤

### 2. 本地皮肤（离线登录专属）
- 特点：仅自己可见，无需联网，无需账号，适合单人游玩
- 操作步骤：
  1.  打开HMCL/PCL2启动器，选择你的离线版本
  2.  进入「版本设置」/「皮肤管理」页面
  3.  选择「本地皮肤」，上传你的皮肤png文件，选择对应的手臂模型
  4.  保存设置，启动游戏即可生效，其他玩家无法看到该皮肤

### 3. 皮肤站皮肤（外置登录通用）
- 优势：无需正版，和你使用同一认证服务器的联机玩家均可看到皮肤，支持自定义披风
- 操作步骤（以LittleSkin为例）：
  1.  登录LittleSkin账号，创建游戏角色
  2.  在皮肤库选择喜欢的皮肤，或上传自己的皮肤，点击「保存至我的衣柜」
  3.  在衣柜中选择皮肤，点击「使用」，绑定到你的游戏角色
  4.  启动器使用对应皮肤站的外置登录，启动游戏后即可生效，联机玩家可正常显示

## 六、Mod安装教程
Mod是拓展游戏内容、优化体验的核心，新手务必严格遵守安装规则，避免游戏崩溃。

### 1. 核心前提（必看，90%的崩溃都源于此）
- **版本必须完全对应**：Mod的游戏版本、加载器类型，必须和你的游戏完全一致（如1.20.1 Forge的Mod，无法用于1.20.1 Fabric，也无法用于1.19.2 Forge）
- **前置Mod必须安装**：大部分Mod都需要前置Mod（如Fabric必须安装Fabric API，Forge很多Mod需要前置），下载页面会明确标注，必须一并安装
- **开启版本隔离**：第三方启动器务必开启版本隔离，每个版本使用独立的文件夹，避免不同版本的Mod冲突
- **Mod文件格式**：Java版Mod均为`.jar`格式，切勿解压，直接使用

### 2. Mod正规下载渠道（安全无病毒）
- Modrinth（推荐，全中文，无广告，安全合规）：https://modrinth.com/mods
- CurseForge（全球最大Mod平台，模组最全）：https://www.curseforge.com/minecraft/mc-mods
- MC百科（国内平台，中文介绍，新手友好）：https://www.mcmod.cn/

> 提醒：切勿从网盘、不知名论坛下载Mod，避免恶意软件、病毒、捆绑安装包。

### 3. 安装步骤
#### （1）启动器一键安装（新手首选，HMCL/PCL2支持）
1.  启动器选择你要安装Mod的版本，进入「Mod管理」页面
2.  点击「下载Mod」，启动器会自动匹配对应版本、对应加载器的Mod列表
3.  搜索你想要的Mod，点击「安装」，启动器会自动下载Mod及其前置Mod，无需手动操作
4.  安装完成后，重启游戏即可生效，可在游戏主菜单的「Mods」选项中查看已安装的Mod

#### （2）手动安装（通用方法）
1.  从正规渠道下载对应版本的Mod文件（`.jar`格式），以及所需的前置Mod
2.  启动器选择对应版本，点击「打开文件夹」/「打开Mods文件夹」
3.  将下载好的所有Mod文件，直接放入`mods`文件夹中，切勿解压
4.  关闭文件夹，回到启动器，启动游戏即可完成安装

### 4. 注意事项
- 不要一次性安装过多Mod，避免内存不足导致崩溃，建议逐步添加测试
- 若游戏崩溃，可查看`crash-reports`文件夹中的崩溃日志，定位冲突的Mod
- 客户端-only Mod（如优化、外观类）仅需客户端安装，服务端无需安装；功能性Mod（如科技、魔法类）客户端和服务端必须同时安装，且版本完全一致

## 七、资源包与光影安装
### 1. 资源包（材质包）安装
资源包用于更改游戏的材质、音效、界面，无需加载器，所有版本通用，不会导致游戏崩溃。
1.  正规下载渠道：MC百科、Modrinth、CurseForge、国内MC社区
2.  安装步骤：
    1.  下载对应游戏版本的资源包，格式为`.zip`，**切勿解压**
    2.  启动游戏，进入「选项」→「资源包」→「打开资源包文件夹」
    3.  将下载的资源包压缩包，直接放入`resourcepacks`文件夹
    4.  回到游戏，资源包会出现在左侧「可用」列表，点击箭头移至右侧「已选」
    5.  点击「完成」，等待加载完成即可生效
3.  注意：资源包有版本兼容性，高版本资源包无法用于低版本，可多个资源包叠加使用，上下拖动可调整优先级。

### 2. 光影安装
光影用于提升游戏画面效果，实现真实的光照、阴影、水面效果，对电脑显卡有一定要求，低配电脑谨慎使用。
1.  核心前提：必须先安装光影支持模组，才能加载光影包，不同加载器对应不同的模组：
    | 加载器 | 推荐光影支持模组 | 特点 |
    |--------|------------------|------|
    | 原版/Forge | OptiFine（高清修复） | 全能型，同时支持光影、优化、材质修复，新手首选 |
    | Fabric/Quilt | Iris Shaders + Sodium | 轻量化，优化更好，帧率更高，新版本首选，兼容绝大多数光影包 |
2.  光影支持模组安装：和普通Mod安装方式一致，启动器一键安装或手动放入mods文件夹，安装完成后需重启游戏
3.  光影包正规下载渠道：MC百科、Modrinth、CurseForge、国内MC社区，主流光影：BSL、Complementary、SEUS、KUDA
4.  安装步骤：
    1.  下载对应版本的光影包，格式为`.zip`，**切勿解压**
    2.  启动游戏，进入「选项」→「视频设置」→「光影包」→「打开光影包文件夹」
    3.  将下载的光影包压缩包，直接放入`shaderpacks`文件夹
    4.  回到游戏，光影包会出现在列表中，点击选择即可启用
    5.  可在「光影设置」中调整光影参数，平衡画质与帧率

## 八、联机教程
Minecraft联机分为多种方式，从简单到复杂依次如下，新手优先选择前两种。

### 1. 局域网联机（同一路由器/同一WiFi下首选）
- 优势：无需额外软件，操作最简单，无延迟，适合和身边的朋友联机
- 前提：所有玩家电脑必须连接同一个路由器/同一个WiFi，关闭电脑防火墙
- 操作步骤：
  1.  主机进入单人世界，按下`ESC`，点击「对局域网开放」
  2.  设置游戏模式、是否允许作弊，点击「创建一个局域网世界」
  3.  游戏左下角会显示「本地游戏已在端口 xxxxx 上开启」，记录端口号
  4.  其他玩家进入游戏，点击「多人游戏」→「直接连接」
  5.  输入主机的局域网IP地址+端口号（格式如`192.168.1.100:25565`），点击加入即可
- 补充：主机的局域网IP，可在电脑设置-网络中查看，Windows也可通过cmd输入`ipconfig`查看IPv4地址

### 2. 启动器自带联机功能（新手跨网联机首选）
HMCL、PCL2均内置一键联机功能，无需公网IP、无需复杂配置，跨网可直接联机，适合和异地好友联机。
- 操作步骤（以HMCL为例）：
  1.  所有玩家使用相同的游戏版本、相同的加载器、完全一致的Mod列表（Mod联机必备）
  2.  所有玩家注册并登录HMCL账号
  3.  主机点击启动器左侧「联机」，创建联机房间，设置房间密码、最大人数
  4.  房间创建成功后，主机进入单人世界，对局域网开放
  5.  其他玩家在联机页面找到主机的房间，输入密码加入房间，即可在多人游戏中看到主机的世界，点击加入即可
- PCL2联机操作逻辑一致，在启动器「联机」页面即可完成，全程无需额外配置

### 3. Mod联机核心注意事项
- 所有联机玩家的**游戏版本、加载器类型、加载器版本、Mod列表（包括前置Mod）必须完全一致**，否则无法进入服务器，甚至直接崩溃
- 仅客户端优化类Mod（如 Sodium、OptiFine）可不一致，功能性Mod必须完全相同
- 建议所有玩家使用同一个整合包，避免出现版本不一致的问题

### 4. 内网穿透联机（进阶，无公网IP跨网联机）
若不想使用启动器联机功能，可使用内网穿透工具实现联机，主流工具：Sakura Frp、FastFrp、Frp，操作步骤如下：
1.  所有玩家安装对应版本的内网穿透客户端，注册账号
2.  主机打开客户端，创建隧道，选择「TCP隧道」，本地地址填`127.0.0.1`，本地端口填游戏局域网开放的端口（如25565）
3.  启动隧道，记录隧道的远程地址和端口（如`xxx.sakurafrp.com:12345`）
4.  主机进入游戏，对局域网开放，确保端口和隧道本地端口一致
5.  其他玩家进入游戏，点击「多人游戏」→「直接连接」，输入隧道的远程地址和端口，点击加入即可

### 5. 第三方服务器加入
- 操作步骤：
  1.  确认服务器的版本、正版验证要求、Mod要求，确保你的游戏版本和服务器一致
  2.  打开游戏，点击「多人游戏」→「添加服务器」
  3.  输入服务器名称、服务器地址（IP+端口，默认端口25565可省略），点击完成
  4.  服务器会出现在列表中，点击「加入服务器」即可
- 注意：正版服务器仅支持微软正版账号登录，离线账号无法进入；Mod服务器需要安装和服务器完全一致的Mod列表。

## 九、常见问题FAQ
1.  游戏启动失败，提示「找不到Java」？
    答：检查Java版本是否和游戏版本匹配，安装路径是否有中文；第三方启动器可在「版本设置-Java路径」中手动选择Java安装目录，或直接使用启动器的一键Java下载功能。

2.  安装Mod后游戏崩溃？
    答：优先检查Mod版本、加载器是否和游戏版本一致；检查是否安装了所有前置Mod；检查是否有冲突的Mod；关闭多余Mod，逐个测试定位问题；适当增加游戏内存分配。

3.  联机时提示「连接超时」「无法连接到服务器」？
    答：检查主机和客户端的网络是否通畅，关闭电脑防火墙、杀毒软件；确认IP地址、端口号输入正确；跨网联机优先使用启动器联机功能，避免内网穿透配置错误。

4.  光影安装后没有光影选项？
    答：确认已正确安装光影支持模组（OptiFine/Iris Shaders），模组版本和游戏版本匹配；光影支持模组已成功加载，可在游戏Mod列表中查看。

5.  皮肤不显示？
    答：正版账号检查皮肤是否在官网上传成功，重启游戏刷新；离线账号皮肤仅自己可见；外置登录账号确认启动器登录的认证服务器和皮肤站一致，角色名和皮肤站角色名完全匹配。
